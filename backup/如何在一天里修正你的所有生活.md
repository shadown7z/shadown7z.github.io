# 如何在一天里修正你的所有生活
在抖音刷到的一篇来自推特的推文：
https://x.com/i/status/2010751592346030461

我感觉这里面的有些东西还挺实用，其实这篇文章主要写的是你该如何坚持的去做一件事。然后作者说了3个有用的技巧：
**1.改变身份**
**2.用厌恶推动自己**
**3.每天做两到三件事**

**我们分开来阐述这三个是什么，怎么做：**
1.改变身份，比如说，你早上不想起床，你的脑子一定是跟起床做对抗，“起床？要不再睡会儿吧？”，“不行，得起床学习啊”，例如前面举的两个例子，可以看到你一直在跟起床做斗争。其实这个时候你只需要改变一些身份即可，“我是一个自律的人，我需要起床”，当你这样告诉自己的时候远比前者的效果强一些，容易起床了一点。我分析了一下，前者是在做斗争，而后者更像是一种心里暗示，并不会像前者一样进行心理斗争。

2.用厌恶推动自己，人对那种美好的理想的东西摸不着，只能想着，没有真切实际的东西，这种坚持不了多长时间。而如果你想着如果，“如果我现在不好好学习，以后可能会跟之前一样去搬货、进厂，像之前一样累，而且还有夜班”，当你把这些坏处跟当下正在经历的事情产生了联系，你就会想着现在还是好好学比较好，用厌恶去推动自己。我分享了一下，可能是因为人们，总是对那些快乐的事情不太在乎，但是对痛苦的事情记忆犹新，所以用厌恶推动自己。

3.每天做两三件事，主要是怕你做多件事情容易崩盘，稳住自己的节奏。每天做两三件事情，坚持一段时间，或坚持一个月左右，再回头看看，你就会发现自己原来已经这么厉害了。感觉这些可以结合着自己的2026年计划进行。

**文章的最后，作者说了这么一句话，大概意思是“如果你现在看到这里了，还不去行动，那么下一年的这个时候你还是会在这里刷到同样的推文”**
